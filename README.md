# Full-stack app with Vue and Express
Created with `Vue`, `TypeScript`, `Sass` & `Express.js`
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Starts the server side
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
